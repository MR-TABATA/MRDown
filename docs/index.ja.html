特徴

ほかのエディタと違うところ。

何が変わったかを読む

保存も、外部からの書き換えも、すべて版として記録。任意の2版を並べて差分を見られます。Git リポジトリなら HEAD も同じ一覧に並ぶので、コミット済みとの差分もそのまま。ワンクリックで復元でき、保存するまで上書きしません。Git は無くても動きます。

安全に表示

DOMPurify でサニタイズし、厳格な CSP でロックダウン。どこから来たか分からない Markdown も、生成されたばかりの Markdown も、そのまま開けます。

AI に上書きされない

ディスク上でファイルが変わればスクロール位置を保ったまま再描画し、その版も履歴に残します。編集中に書き換えられたら、どちらも捨てません。「最後の保存 / ディスク / 自分の編集」を3列で並べ、双方が同じ行を別々に直したところだけを衝突として示します。黙って上書きすることも、黙って読み直すこともしません。

.md リンクでアプリ内遷移

ローカルの Markdown へのリンクはブラウザに出ず、その場でアプリ内に開く。相互リンクされた文書群を、アプリを出ずに辿れます。外部リンクはブラウザ、見出しアンカーはスクロール。

ひととおり、揃っています

  • 編集 & ライブプレビュー

    左で書いて右で即確認の分割ビュー。⌘S で保存、幅はドラッグで調節。

  • 検索 & 置換

    ⌘F で文書内を検索。正規表現・大小区別・単語単位。プレビューではタグをまたいでマッチ。

  • 複数文書を並べて

    開いた文書をサイドバーに一覧。クリックで切替、× で閉じる。次回起動時に復元。

  • フォルダをまとめて

    フォルダ内の Markdown を左のツリーに一覧。サブフォルダは展開／折りたたみ。

  • アウトラインで飛ぶ

    見出しの目次を本文の右に独立した列で。読んでいる位置が自動でハイライトされ、編集中もライブ追従。⌘2 で開閉（編集中は既定で畳んでプレビューに幅を譲ります）。設定で左サイドバー内にも置けます。

  • 書き味の効くエディタ

    Enter でリスト自動継続、Tab でインデント、括弧は自動で閉じる。URL を貼ればリンク、画像を貼れば保存して挿入。

  • 外観をカスタマイズ

    アクセント色・背景色・文字色、本文フォントとサイズ。背景色はウィンドウ全体に反映。

  • 表・コード・図

    テーブル、シンタックスハイライト付きコード、引用、ローカル画像、Mermaid 図。先頭の YAML frontmatter は畳んだメタ情報カードに。

  • 下書きを失わない

    編集中の内容も未保存の無題文書も自動保持。終了やクラッシュでも次回起動時に復元。

  • 数式・脚注・タスクリスト

    KaTeX で数式を組版、脚注は末尾に相互リンク付きで、タスクリストはチェックボックスで描画。

  • HTML / PDF で書き出す

    ⌘⇧E で自己完結した 1 枚の HTML に。⌘P で印刷ダイアログから PDF 保存。

プレビルド版をダウンロード

最新版をダウンロード

macOS 版（Apple Silicon / Intel、macOS 10.15 以降）を配布。Apple Developer ID で署名・公証（notarization）済みなので、ダウンロードしてそのまま起動できます。

または、ソースからビルド

# リポジトリを取得
$ git clone https://github.com/MR-TABATA/MRDown.git
$ cd MRDown

# 依存をインストール
$ npm install

# 開発モードで起動
$ npm run tauri dev

# 配布用にビルド
$ npm run tauri build
